ol-ext: Contour grid data

Tips：Import D3 before using ol.source.Contour

ol/source/Contour accepts grid input:
{ x:number[][], y:number[][], z:number[][] }

In this example x=longitude, y=latitude, z=value. Data is treated as EPSG:4326 (no coordinate transform).
